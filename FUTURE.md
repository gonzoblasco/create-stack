# Proyectos futuros

Este documento lista ideas y posibles expansiones del ecosistema `@gonzoblasco/create-stack` que **no están en el alcance actual**.

---

## Nuevos stacks

- `tanstack` — Next.js + TanStack Query + TanStack Start
- `astro` — Astro + Content Collections + integraciones
- `cli` — Scaffolder de CLIs en Node.js
- `remix` — Remix Run
- `sveltekit` — SvelteKit

## Expansión del ecosistema

- Sub-agentes de OpenClaw que ejecuten tareas de scaffolding o migración automáticamente.
- Plugins de IDE/extensiones para generar proyectos desde UI.
- Integración con registries de componentes (shadcn/ui, etc.).

## Otras ideas

- `create-stack` como CLI global instalable
- Modo interactivo sin argumentos (seleccionar stack + nombre con prompts)
- Template versionado (elegir versión de Next.js, etc.)
